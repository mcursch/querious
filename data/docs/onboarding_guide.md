# New Employee Onboarding Guide

## Week 1 Checklist

- [ ] Complete I-9 and W-4 with HR
- [ ] Set up company email (@acmeoutfitters.example)
- [ ] Enroll in benefits (30-day window from start date)
- [ ] Complete security training (mandatory, see Security Policy)
- [ ] Request system access from IT (ticket required for each system)
- [ ] Meet with manager for 30-day plan discussion

## Systems Access

| System | Request Via | Approver |
|---|---|---|
| ERP (NetSuite) | IT Help Desk ticket | Department Head |
| CRM (Salesforce) | IT Help Desk ticket | Sales Manager |
| Support platform | IT Help Desk ticket | Support Manager |
| GitHub | IT Help Desk ticket | Engineering Lead |
